指令管理与快捷发送

强大的指令管理系统，支持常用指令的增删改查。可保存历史发送记录，支持 TXT/HEX 格式，配置发送间隔时间实现定时发送。配合快捷键配置，一键快速发送，大幅提升调试效率。

指令管理器 支持创建、编辑、删除常用指令，可为每个指令设置名称、内容和格式。

快捷键配置 支持为常用指令绑定快捷键，Ctrl+数字键快速发送，让调试更便捷。

定时发送 支持配置发送间隔时间（毫秒级），实现循环定时发送，方便进行压力测试。

多主题样式切换

LTSerialTool 支持明亮/暗黑等多种主题样式切换，基于 AtlantaFX 主题框架，界面简洁美观。同时提供样式文件模板，支持用户自定义 CSS 样式文件来定制独特的 UI 风格。

主题切换 支持一键切换明亮/暗黑主题，适应不同的工作环境和个人偏好。

自定义样式 提供样式文件模板，用户可编辑 CSS 文件创建个性化主题，打造独特的界面风格。
